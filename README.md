# YOLO Object Detection Comparison

Projeto desenvolvido para comparar os modelos YOLO Nano, Small, Medium, Large e Extra Large na detecção de objetos em imagens com alta densidade de elementos.

## Tecnologias utilizadas
- Python
- Google Colab
- OpenCV
- Ultralytics YOLO

## Modelos testados
- YOLO Nano (n)
- YOLO Small (s)
- YOLO Medium (m)
- YOLO Large (l)
- YOLO Extra Large (x)
